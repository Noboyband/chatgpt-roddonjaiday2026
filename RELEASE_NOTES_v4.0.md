# Roddonjai Day 2026 - Version 4.0.0

Release date: 2026-07-17

## Included

- Unified white, orange, and navy visual theme across the stage and mobile pages.
- Branded circular 3D interaction button with the orange Roddonjai heart mark.
- Updated collaboration message shared across both pages.
- Mobile heart animations with shuffled orange, blue, red, and yellow colors.
- Text-rendered heart symbols for reliable multi-color display on mobile browsers.
- Mobile page scrolling support so the score remains reachable on short screens.
- Heart interaction capped at 500 clicks per device.
- Restored background music defaults, visible music controls, and inline mobile playback support.
- Simplified stage QR Code presentation with responsive desktop and mobile layouts.
- Custom domain deployment at `https://rdj.noboyband.com`.
