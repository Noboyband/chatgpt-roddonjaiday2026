# Roddonjai Day 2026 Live Hearts

เว็บไซต์: https://noboyband.github.io/chatgpt-roddonjaiday2026/

หน้าเว็บกิจกรรมสำหรับจอเวทีและมือถือผู้ร่วมงาน โดยหัวใจและเสียงเอฟเฟกต์เกิดเฉพาะบนมือถือแต่ละเครื่อง

## เปิดใช้งาน

1. เชื่อมต่อคอมพิวเตอร์จอเวทีและมือถือกับ Wi-Fi เดียวกัน
2. รัน `npm install` (ครั้งแรก) และ `npm start`
3. เปิด URL แบบ Network ที่แสดงในหน้าต่างคำสั่งบนจอเวที
4. ผู้ร่วมงานสแกน QR Code บนจอเพื่อร่วมส่งหัวใจ

เสียงเอฟเฟกต์และแอนิเมชันหัวใจจะแสดงเฉพาะมือถือที่แตะหัวใจ และมีเสียงปรบมือ/เสียงเฮเพิ่มเติมทุก 10 ดวง
ความเร็วในการแตะมีเสียง 4 ระดับ: จังหวะปกติ, กำลังเร่ง, ติดเทอร์โบ และรัวสุดพลัง โดยระดับสุดท้ายต้องแตะอย่างน้อย 12 ครั้งภายใน 1.4 วินาที

ตัวนับ 1,000 ดวงจะถูกจำแยกไว้ในมือถือแต่ละเครื่อง และไม่มีการส่งหัวใจไปยังจอเวที
