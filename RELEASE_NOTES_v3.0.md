# Roddonjai Day 2026 - Version 3.0.0

Release date: 2026-07-17

## Included

- Custom domain deployment at `https://rdj.noboyband.com`.
- QR Code linking directly to `https://rdj.noboyband.com/play.html`.
- Responsive stage screen with mobile QR Code support.
- Improved desktop alignment for the logo, stage message, LIVE indicator, and QR Code.
- Mobile heart interaction capped at 500 clicks per device.
- Four speed levels with distinct sound effects and colored heart animations.
- Shared low-volume MP3 background music on both pages.
- Responsive favicon set for desktop, Android, and Apple devices.
